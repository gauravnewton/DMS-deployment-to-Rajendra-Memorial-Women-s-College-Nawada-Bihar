  1. Introduction

    "Digital Data Management System" is a complete solution for managing all type of document related activities occurs in an organization. Some of the highlighting features are as follows......

    • Security at two level (admin level and user level).
    • Creation of user
    • Seperate permission for seperate user.
    • Quick upload documents from local drive.
    • Wide range of supported documents including in quick upload section including 'pdf','dox','docx','xls','xlsx','jpg','jpeg','png','bmp','titf' etc.
    • Scan and upload feature enables user to scan any document via any connectd scanner to system in 'png','jpeg' and 'titf' format and save it to database.
    • Smooth and customized searching for documents.
    • Searching customization includes search via file name, file department, file category, file type, file creator, date of creation or combination of them.
    • It also provides the facility to backup the recent database to local drive as well as you can also restore a recent backedup file to the system in case of system failure.
    • In setting it provides the facility to change the application's theme, application's resolution and the default databse backup path.
    • In all this product is being used for commercial purpose hence it requires to purchase a licence version from IT Care Gaya with a valid product key. Note that while activating this product your system must be connected to a working internet connection.
    • Keyboard shortcuts for every action.



  2. Get Started

    In order to effective use of this product you must have to follow some setups for first time as provided below..

    • After successfull installation of this product you have to register a new admin by providing some details along with admin id and passwords, which are later used for login into this product.
    • Now Login with yur admin id and password, after successfull login you have to create some(at least one) user by providing their details.
    • Grant appropriate permission to that user.
    • Install a proper scanner driver for effective working of scan and upload feature.



  3. File Menu

    File menu contains a clear cache option, which clear local temporary file created by software while working for a long time. Note that while clearing cache the operation may be blocked by any other running application which are currently accessing the local file. For more information on clearing cache consult to app vendor.



  4. Admin Menu

    Admin menu only activated when an admin logged in or any user having corresponding permission. Admin menu contains 4 different actions...

    User creation

    Admin or any user having permission to add new user can add a new user from this menu by providing user's details.



    Edit user

    Admin or any user having permission to add edit user can edit a user from this menu by providing choosing an existing user from combobox.



    Add department

    Admin or any user having permission to add department can add a department from this menu.



    Add Category

    Admin or any user having permission to add category can add a category from this menu.



  5. Report Menu

    This menu show a detailed report about uploaded file, downloaded file or viewd file by user for selected date.



  6. Documents Menu

    Document menu also divided into three category as follows...



    Quick upload

    A user can upload any file using this menu. User have to specify the file department and category before uploading file to database.



    Sacn and upload

    A user can scan any document via any connected scanner to the system. First user has to select a available scanner from listbox then he/she has to specify file category,department and choose any file format then start scan after scanning save the file to data base.



    Searching

    A user can search and download any available file if he/she has corresponding permission. Searching of file can be done with a wide verity of parameters like by name, by file type, by file creator, by date of creation,by department and by category.



  7. Backup and Restore Menu

    User if authorized then they can create a backup of currently running database for data safety in case of system error or system migration, the created backup can be found at the path specified in setting and later can be restored by this menu.



  8. Setting Menu

    In setting user can specify the app theme , screen resolution and even the databas ebakup path. A restart may needed for reflect proper effect after saving settings.



  9. Product Activation Menu

    To commercial use of this application user has to activate it by providing a valid 30 digit product provided at the time of roduct purchasing.
